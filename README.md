## 乐理
- 分母：一个拍由哪种音符构成
- 分子：一个小节有多少拍
- 1/4音符是由8个32分音符组成的，是音符的一种，1/4拍是一种节拍类型，代表一个4分音符为1拍，1小节有1拍
